Semantic Tags in HTML

Semantic HTML or semantic markup is HTML that introduces meaning to the web page rather than just presentation. For example, a <header> tag indicates that the enclosed text is a header. A <footer> tag indicates that the enclosed text is a footer. A <article> tag indicates that the enclosed text is an article.

HTML Tables

HTML tables are defined with the <table> tag. A table is divided into rows with the <tr> tag. A table row is divided into data cells (or columns) with the <td> tag. The <th> tag is used to define a header cell in a table.
